# Resume-Builder-Website
A web-based resume builder where users can enter personal information, choose from templates, and preview their resume in real time. Supports exporting resumes as PDF with clean, print-ready formatting. Built as part of a Web and Security Technologies (WST) course project using HTML, CSS, and JavaScript.

## Features
- User Login and Signup
- Dashboard to manage resumes
- Services page
- Contact page
